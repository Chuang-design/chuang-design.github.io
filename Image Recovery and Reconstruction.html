Image Denoising and Restoration

Images are widely used carriers of information. However, during acquisition, transmission, or storage, images are often affected by noise. Effectively removing noise and recovering the original visual content is a critical problem in computer vision.

Noise Generation

Given an original image I ∈ RH×W×C, we apply a binary noise mask M ∈ RH×W×C (values in {0,1}) to generate a corrupted version X = I ⨀ M.

Image Restoration Methods

1. Mean Filtering

Mean filtering replaces each pixel value with the average of its neighbors, using a predefined kernel. This helps smooth out noise and create a more consistent region.

  • Arithmetic Mean Filter
  • Harmonic Mean Filter
  • Geometric Mean Filter

2. Median Filtering

Median filtering is a nonlinear method based on sorting statistics. Each pixel is replaced by the median of its local neighborhood, effectively removing isolated noise while preserving edges.

3. K-Nearest Neighbors (KNN) Filtering

This method compares each pixel with its surrounding neighbors (within an N×N window) using Euclidean distance. The pixel is reconstructed from the average of the K most similar neighbors based on color similarity.

4. Deep Image Prior (DIP)

Deep Image Prior is a neural-network-based method that does not require pre-training. By leveraging the network’s natural bias toward structured images, DIP can recover clean content from noisy inputs.

In this project, we use an encoder–decoder architecture with downsampling (via convolution) and upsampling (via transposed convolution). Early stopping is used to prevent overfitting to noise.
